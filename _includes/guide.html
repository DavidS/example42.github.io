What

The [DevOps] Guide to Puppet, Universe, and Everything is a wannabe holistic documentation project composed by:
  • A Book, about Puppet, and Puppet Universe.
    It will be published when reviewed, refined and good enough for a first edition.
  • A set of Puppet architectures: documented examples of Puppet code and data layouts to test or used as kickstasters for new projects.
  • A modular Slides deck with tutorials on different Puppet topics.
    They will be free to be used for Puppet workshops.
  • A practical Puppet Universal Reference with the info and the links that matter.

How

We know something about the pains of writing a good book and here we plan to do something more than that.
This would take us a lot of time and we wonder if this is something actually needed or worth, for others.
If you have ever appreciated or used the Puppet works of Example42, or if you think that you may find useful this project, support our crowdsourcing campaign on IndieGogo.
If you want to actively contribute to the project check where everything is shared.

When

  • If the Indiegogo campaign is successful we plan to publish the first edition of the book in summer of 2016.
  • We have already some samples of Puppet architectures to share, but we intend to update, redesign and document them in autumn 2015.
  • The slides are going to be grouped in different modular sections. We plan to release a first complete set by the end of winter 2015.
